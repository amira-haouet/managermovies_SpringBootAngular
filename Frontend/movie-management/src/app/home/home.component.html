Description

Smart Doctor est une application Web créée dans le but de faciliter la gestion des medecins et leurs spécialités au sein des cliniques ou des hôpitaux ; elle permet effectuer les opération CRUD sur les médecins et les spécialités .
En utilisant en Front-End le framework Angular , et pour le style le framework CSS Bootstrap et en Back-End le framework Java Spring Boot.
